#  Student Mentor

A Chrome extension that acts as a **mentor**, not a solution giver, while solving coding problems on LeetCode.

## Features (Current)
- Reads problem statement directly from the screen
- Popup chat UI
- Blocks solution requests
- Redirects out-of-context questions

## Tech Stack
- Chrome Extension (Manifest v3)
- Vanilla JavaScript
- FastAPI 
- LLM-based mentor 

## Philosophy
This tool guides students using hints and questions instead of giving direct answers.

## Status
In active development
